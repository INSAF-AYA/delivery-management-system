1- Project Overview
SwiftShip is a web-based delivery management system designed to help logistics companies manage clients, agents, drivers, shipments, and invoices efficiently.
The system centralizes operations, reduces manual work, and provides a clear dashboard for decision-making.

2- Objectives
Simplify delivery and shipment management
Manage users (Admin & Agents) with role-based access
Track shipments and delivery status
Generate invoices
Improve operational efficiency
